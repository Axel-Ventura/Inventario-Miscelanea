# Sistema de Gestión de Inventarios para Misceláneas

## Descripción
Aplicación web orientada a la gestión de inventarios para misceláneas y pequeños comercios.

## Objetivo
Facilitar el control de productos, existencias y movimientos de inventario mediante una solución web accesible.

## Tipo de sistema
Aplicación web.

## Estado del proyecto
Fase de análisis y definición de arquitectura.
