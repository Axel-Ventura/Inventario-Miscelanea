Backend - Tecnologías

1. Node.js + Express
Para crear el servidor y manejar las rutas.

2. MySQL / MariaDB
Base de datos donde guardaremos la información.

3. Prisma ORM
Para trabajar con la base de datos de forma más fácil.

4. JWT (jsonwebtoken)
Para manejar el login y proteger las rutas.

¿Qué hace cada una?
Node.js + Express → El motor del backend. Recibe peticiones y envía respuestas.

MySQL → Donde se guardan los datos (usuarios, productos, etc.).

Prisma → Nos ayuda a escribir menos código SQL. Convierte JavaScript a consultas de base de datos.

JWT → Crea "tickets" seguros para saber quién está usando la aplicación.
